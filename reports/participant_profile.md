## project_idea

### concept
Health AI Maturity Model Assessment + RFP Tool — an agent-guided platform that helps CIOs assess their health system's AI infrastructure readiness and generate procurement-ready outputs.

### problem_space
CIOs at health systems are making high-stakes AI infrastructure procurement decisions without a structured framework. Existing maturity models (like HIMSS) address the technology/clinician intersection but ignore AI infrastructure specifics — regulated data proximity, inference latency, on-prem vs. cloud tradeoffs. This tool fills that gap.

### proposed_approach
The full vision: CIO answers questions across 7–8 maturity domains, uploads internal documents, and the system pulls publicly available data on the health system. Outputs include a maturity score + narrative report, a board/executive-ready PPTX, and refined RFP language with procurement considerations. Likely no-code implementation using n8n. For the 7-day bootcamp, scope is reduced to building one central agent (TBD which component).

### success_criteria
A working single agent that demonstrates the core assessment loop end-to-end — enough to validate the concept and establish the pattern for the full multi-agent system.
